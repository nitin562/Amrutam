
# Amrutam Task app 
This is brief description of setuping and runing the Amrutam design created by Nitin Dabas
## Run Locally

Clone the project

```bash
  git clone https://github.com/nitin562/Amrutam.git
```

Go to the project directory

```bash
  cd Amrutam
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
